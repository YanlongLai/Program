### JavaScript
 - Function
 ```javascript

 function FuncA(){
         return new function(){
                     alert('Is it strange?');
                         }
 }
  
  var f1 = FuncA;
  var f2 = f1();
  f2(); //Is it strange?
 ```
 在C#中「function is a part of Object」，也就是說C#在語義上將function與Object定義為從屬關係，對於C#或Java程序員來說，這種想法已經深入骨髓了。在JS中，function IS A KIND OF Object。
